# Shutter-mall（快门商城）

为了学习编写的简单商城系统。

本仓库为后端仓库。

前端（用户端）：[https://github.com/st1020/shutter-mall-web](https://github.com/st1020/shutter-mall-web)

前端（管理端）：[https://github.com/st1020/shutter-mall-web-admin](https://github.com/st1020/shutter-mall-web-admin)

## 使用的技术栈

后端

* Java
* Spring Boot
* JPA
* JWT
* MySQL

前端：

* Vue
* Vue Router
* Pinia
* Vant
* Naive UI

## 功能支持

* 支持用户登陆注册
* 支持基本的权限管理
* 支持商品浏览，购物篮，购买下单
* 支持订单管理，用户管理，商品管理等功能
